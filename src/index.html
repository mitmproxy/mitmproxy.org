mitmproxy

An interactive console program that allows traffic flows to be intercepted, inspected, modified and replayed.

mitmdump

Think tcpdump for HTTP - the same functionality as mitmproxy without the frills.

libmproxy

A library for implementing powerful interception proxies.

A privacy analysis project, built using the libmproxy library. An example of a high-performance server-side application of mitmproxy.

tamper

A mitmproxy based devtools extension that lets you edit remote files locally and serve them directly to Chrome.
